## 갱신 규칙

1. **복붙 금지**: Notion 기획·Figma 토큰은 항상 MCP/Playwright로 실시간 조회해 반영한다. 스냅샷을 손으로 복사하면 드리프트가 발생한다.
2. **코드가 진실**: 코드와 이 문서가 어긋나면 코드를 기준으로 문서를 맞춘다.
3. **화면 파일 생성 시점**: 각 화면의 세부 노트(`screens/<id>.md`)는 구현 직전 Figma Description Note에서 생성한다. 미리 만들지 않는다.
4. **ADR 연동**: 구조적 결정이 생기면 `docs/adr/`에 ADR을 추가하고, 이 허브의 해당 파일에서 참조 링크를 건다.

## 참조 우선순위

```
prd-v1.md (PO 확정 스펙)
  > Figma (레이아웃·토큰)
    > Notion 회의록 (맥락·배경)
      > 이 허브 나머지 파일 (파생 요약)
```

PRD v1에 명시되지 않은 사항은 PO에게 확인 후 `open-issues.md`에 기록한다.
